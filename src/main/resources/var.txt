var keyword

Java 10 ile var keyword'ü geldi. Tip belirtmeden değişken yazmamızı sağlar. Diğer adı "local variable type inference". Sadece local değişkenler için kullanılır. for looplarının içinde, lambdalarda, try-with-resources da kullanılabilir. Compiler compile time da tipini atanan değere göre tahmin eder. Tipi değiştirilemez. Runtime da ise değeri farklı tipler olabilir, tek şartla atanan tip ilk tipin içine sığmalıdır. 
- var keyword ü tanımlandığı anda değer ataması yapılmalıdır, aksi taktirde kod derlenmez. Çünkü değere göre tip ataması o anda yapılmalıdır. 
- var ile birden fazla değişken tanımı aynı satırda yapılamaz.
- var a = null derlenmez, çünkü hangi nesnenin null'ı, type'ı tahmin edemez. Ama şu satur derlenir. var o = (String)null;
- var değişken adı olabilir! var var = "var"; derlenir!
- var class, interface, or enum adı olamaz.
- var reserved word değildir çünkü eski kodlarda kullanılmış ise ilgili kodlar kırılır. 

- A var is used as a local variable in a constructor, method, or initializer block.
- A var cannot be used in constructor parameters, method parameters, instance variables, or class variables.
- A var is always initialized on the same line (or statement) where it is declared.
- The value of a var can change, but the type cannot.
- A var cannot be initialized with a null value without a type.
- A var is not permitted in a multiple-variable declaration.(var winter = 12, cold; doesn't compile)
- A var is a reserved type name but not a reserved word, meaning it can be used as an identifier except as a class, interface, or enum name.


var squirrel = new Object();   // geçerlidir. Nesne atanabilir.
